# Кастомный элемент Select Search на основе UI библиотеки Admiral.

1. @admiral-ds/react-ui
2. @admiral-ds/flags
3. @admiral-ds/fonts
4. @admiral-ds/icons

1. styled-components > 5.1.0
2. react > 16.0.0
3. react-dom > 16.0.0
